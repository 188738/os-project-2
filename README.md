# os-project-2
This project simulates a bank operation with three tellers and 50 customers. It demonstrates concurrent programming and thread synchronization using Semaphores and Monitors in Java. The simulation ensures thread safety when accessing shared resources

Ensure you have the Java Development Kit (JDK) installed and configured on your machine.

javac *.java run that command

java BankSimulation run this command

The simulation initializes 3 Teller threads and 50 Customer threads
